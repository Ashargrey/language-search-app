# 🚀 Tech Stack Directory

A modern and responsive web app that allows users to search and explore different programming languages, frameworks, libraries, and tools.

---

## 📌 Features

- 🔍 Live search (real-time filtering)
- 🎨 Clean and modern UI
- 📱 Fully responsive design
- ⚡ Fast and lightweight (Vanilla JavaScript)
- 🧠 Search by name or description

## 🛠️ Tech Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
